## Setup

1. Clone the repository

```bash
git clone https://github.com/ZahraJamshidii/photo-gallery.git
```

2. Install dependencies

```bash
npm install
```

3. Create a `.env` file in the project root and add:

```env
VITE_PEXELS_API_KEY=YOUR_PEXELS_API_KEY
```

4. Run the project

```bash
npm run dev
```
